# AnimeEdit AI

AI-powered After Effects plugin for automated anime-style video editing and enhancement.

## Project Structure

```
├── docs/           - Documentation
├── plugin/         - After Effects extension (CEF/CSXS)
│   ├── CSXS/      - Extension manifest and configuration
│   ├── host/      - Host-side scripts (JSX)
│   ├── assets/    - UI assets (icons, HTML, CSS)
│   └── src/       - Extension source code
├── backend/        - Python AI/ML backend
│   ├── app/       - Application logic
│   └── tests/     - Backend tests
├── prompts/       - LLM prompt templates
└── tests/         - Integration and E2E tests
```
